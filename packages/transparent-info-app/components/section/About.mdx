# 关于本项目

## 起源

在十二月份在群里就有人号召 不要去武汉，再后来信息就消匿了一段时间。 直到快过年了 忽然又到处见到疫情的信息，可能是我不怎么关注新闻。但是正确的疫情的信息传播 能让更多的人提前做好准备、不会盲目恐慌。我今年过年一个人呆杭州，大年初二的时候疫情貌似开始比我预估的严重，于是考虑做点什么，之前技术圈也有一些开源的信息的收集平台。但是作为一个曾经的区块链从业者，我有不同角度的思考。 于是就有了此项目。

本项目的代码部分已经初步完成，剩下的就是信息的收集、整理 希望感兴趣的盆友一起加进来贡献。
